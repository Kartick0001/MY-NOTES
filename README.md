## Linux fundamentals
Basic commands for beginner 
1) To know the source name : "whoami"
2) To know the location in which you are present now : "pwd"
3) To check the date of today : "date"
4) To know only date : "date +%D"
5) To know only time : "date +%T"
6) To create a file : "touch filename.txt" (here is an example of .txt file)
7) To create a folder : "mkdir foldername" (here mkdir stands for makedirectory)
8) To see the file content : "cat filename"
9) To change the directory (move fromm one directory to another) : "cd nameofdirectory"
10) To delete any file : "rm filename.txt" (here is an example of .txt file)
